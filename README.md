# 🦅 Eagle-Ping

Un mini-jeu de Ping-Pong ultra-fluide conçu pour les **Farcaster Frames** et optimisé pour le mobile.

## 🚀 Caractéristiques
- **Mobile First :** Contrôles tactiles optimisés pour les écrans 120Hz (comme l'Oppo Find X3 Pro).
- **Fast Deployment :** Déployé instantanément sur Vercel.
- **Eagle Ecosystem :** Fait partie de la suite de mini-jeux développée par @eagle-pi.

## 🕹️ Comment jouer ?
Glissez votre doigt verticalement sur l'écran pour déplacer l'Aigle et battre le Robot.

## 🛠️ Tech Stack
- HTML5 Canvas
- JavaScript (Vanilla)
- CSS3
